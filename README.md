P# Desafio 2 — Catálogo com API

## Visão geral

Neste desafio você irá construir uma **aplicação responsiva que consome uma API externa** e exibe um catálogo de produtos.

OBS IMPORTANTE!! : DURANTE O DESENVOLVIMENTO A API ESTAVA APRESENTANDO UM ERRO DE INVALID SSL, POR ESSE MOTIVO TEMOS INFORMAÇÕES DE PRODUTOS E CATEGORIAS TAMBEM MOCKADAS... QUANDO VOCE FOR REALIZAR OS TESTES, PODE SER QUE A API TENHA VOLTADO E HAJA ALGUMA CONVERGENCIA DE INFORMAÇÕES, CASO ACONTECER ISSO, SE FOR POSSIVEL ME AVISE QUE EU FAÇO AS ADAPTAÇÕES E CORREÇÕES NECESSARIAS...
